# Learning Linux Commands

## pwd
shows current directory path

---

## ls
shows files and folders in current directory

---

## ls -l
shows detailed information of files and folders

---

## ls -lt
shows files according to modified time

---

## ls -ltr
shows files according to modified time in reverse order

---

## cd
used to move between folders

---

## cd ..
moves one folder back

---

## cd ~
moves to home directory

---

## mkdir
creates new folder

---

## touch
creates empty file

---

## rm
deletes file

---

## mv
moves or renames file/folder

---

## cp
copies file

---

## cat
shows complete file content

---

## less
used to read large files page by page

---

## head
shows first lines of file

---

## tail
shows last lines of file

---

## nano
used to create and edit file inside terminal

CTRL + O = save file

CTRL + X = exit nano

---

# Path Understanding

## absolute path

---

## relative path

# Important Learning

- touch creates files
- mkdir creates folders
- cd only works for folders
- files and folders are different in linux
- cd .. moves one folder back
- cd ~ moves to home directory
